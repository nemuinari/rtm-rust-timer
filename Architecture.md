# RTM Architecture Guide

このドキュメントでは、Rust と egui を使用したデスクトップタイマーアプリ `rtm` がどのようなステップで構成され、各モジュールがどう相互作用しているかを解説します。

---

## 🏗️ 開発の 4 ステップ

理解を深めるために、以下の 4 つのフェーズに分けて構造を把握します。

### Step 1: プロジェクトの「外装」 (main.rs)

**役割:** OSに対して「どんなウィンドウを出すか」を伝え、アプリを起動する。

- **NativeOptions:** ウィンドウのサイズ、装飾（タイトルバー）、透過性、最前面表示などの「初期設定」を定義します。
- **Font Setup:** `egui` は標準では日本語や特殊なデジタルフォントを持たないため、起動時に `.ttf` ファイルをメモリに読み込み、アプリ全体のデフォルトフォントとして登録します。
- **Main Loop:** `eframe::run_native` が呼ばれると、OSからのイベント（クリックやキー入力）を待ち受ける無限ループが開始されます。

### Step 2: 静的な「レイアウト」 (app.rs - View)

**役割:** 画面上にパーツ（ラベルやボタン）を配置する。

- **Immediate Mode GUI:** `egui` は毎フレーム（秒間約60回）画面をすべて書き直します。
- **Layout Units:** `CentralPanel`（中央領域）、`Vertical`（垂直並び）、`Horizontal`（水平並び）などのコンテナを使って、UIの骨組みを作ります。
- **Styling:** `RichText` や `Color32` を使い、ダークモードに映える配色やサイズを指定します。

### Step 3: タイマーの「核」 (timer.rs - Logic)

**役割:** UIとは切り離された、純粋な時間の計算。

- **State Management:** 「実行中か？」「停止中か？」というフラグと、経過時間を保持します。
- **Precision:** `std::time::Instant`（高精度タイマー）を使用します。
  - 描画ループの速度に依存せず、OSのシステムクロックから「開始時からの正確な差分」を計算することで、1ミリ秒の狂いもない計測を実現します。

### Step 4: 相互作用の「循環」 (app.rs - Controller)

**役割:** ユーザーの入力をロジックに伝え、結果を画面に反映する。

- **Event Handling:** - ボタンクリック (`.clicked()`) や キー入力 (`ctx.input()`) を検知。
  - 検知したら `timer.toggle()` などのメソッドを呼び出し、内部データを書き換えます。
- **Repaint Request:** タイマー稼働中のみ `ctx.request_repaint()` を呼ぶことで、数字が変化するたびに画面を強制的に書き換え、スムーズなアニメーションを実現します。

---

## 🔄 データフロー図

1. **Input:** ユーザーが [S] キーを押す。
2. **Update:** `app.rs` が入力を検知し、`timer.rs` のフラグを `running` に変える。
3. **Logic:** `timer.rs` が現在の時刻と開始時刻の差分を計算し、経過時間を算出。
4. **Draw:** `app.rs` が算出した経過時間を文字列に整形し、画面に描画する。
5. **Loop:** 次のフレームで 1 に戻る。

---

## 🛠️ 技術スタックの役割分担

| モジュール       | 技術 / クレート    | 主な役割                                   |
| :--------------- | :----------------- | :----------------------------------------- |
| **Entry Point**  | `eframe`           | OSウィンドウの制御、ライフサイクル管理     |
| **UI Framework** | `egui`             | 即時モードによる描画、ウィジェット配置     |
| **Time Logic**   | `std::time`        | 高精度な時間計測                           |
| **Formatting**   | `chrono`           | 秒数を `00:00:00` 形式の文字列へ変換       |
| **Assets**       | `image` / `winres` | アイコンのパースと実行ファイルへの埋め込み |
